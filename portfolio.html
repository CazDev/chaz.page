Eclidean Algorithm

Completed November 2018
For University

Euclid's Algorithm in java. Outputs table format.
Useful for large numbers and fast automated calculation.
Uses the input function format GCD(a, b). a and b are integer inputs.
The function input is hard coded so the input must be given in main() as:
gcdTable(a, b); // function to solve for greatest common deviser in table form
gcd(a, b); // function to solve for greatest common deviser in function form
gcdExtended(a, b); // extended function to solve for the x, y coefficients of a and b

Audio ShadowPlay

Temporarily stores all audio to be played back later.

View details

Project Title

Completed March 2019
For Daniel Rieck

ShadowPlay but for audio.
Commissioned by Daniel Rieck.

How it works
Temporarily stores all audio. When a button is pushed past audio is retrieved from the temporary audio and saved. This gives the same effect as shadow play by NVIDIA but for audio.
Features
Customisable save intervals, High Quality Audio, Variable sampling rate, Data is stored in %appdata%/ASP/
